# FreeCodeCamp JavaScript Practice

This repo contains my JavaScript exercises and mini-projects while working through FreeCodeCamp (and related practice as I learn).

## What’s inside
- **DOM & Events**: event listeners, class toggles, UI interactions
- **Algorithms**: small problem-solving challenges
- **Array Methods**: map/filter/reduce practice
- **Mini Projects**: small apps that combine multiple concepts

## Folder structure
dom-and-events/
algorithms/
array-methods/
mini-projects/

Each folder contains one or more exercises/projects, usually with a short note at the top of the file explaining the goal.

## How I’m using this repo
- Keep solutions organized by topic
- Commit regularly to track learning progress
- Refactor older solutions when I learn a better approach
